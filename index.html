Summary: Latitude vs. X


This project focused on trying to identify trends within weather data based on the given latitude. To accomplish this, over 500 random latitudes and longitudes were generated and then the nearest city to that point was stored. The city weather data was pulled from OpenWeatherMap API to create a dataset.

After creating the dataset, Matplotlib was used to plot different weather factors versus latitude. The weather factors that were analyzed included: max temperature, wind speed, humidity, and cloudiness. This website provides the dataset and the visualizations that were created for each factor. An explanation of the results and descriptions of trends is contained in the 'Plots' dropdown.
